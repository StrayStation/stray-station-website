# Stray Station

**Where curiosity meets code.**

My AI laboratory for experimental apps, creative exploration, and learning to build the future—one project at a time. A playground for the professionally curious.

## About

Stray Station is a personal portfolio and playground showcasing AI experiments, creative applications, and learning projects. Built as a learning journey to explore modern web technologies and AI development.

## Projects Featured

- **Lucky Stray** - Intelligent NC Cash 5 lottery number selector
- **Fractal Art Generator** - Create personalized fractal artwork from meaningful dates
- **Break** - Therapeutic app for safe emotional release
- **Crimeology** - Interactive crime-solving scenarios

## Technology Stack

- HTML5
- CSS3 (with modern gradients and animations)
- Vanilla JavaScript
- Responsive Design

## Local Development

1. Clone this repository
2. Open `index.html` in your browser
3. That's it! No build process required.

## Deployment

This site is deployed on Netlify at [www.straystation.com](https://www.straystation.com)

### Deploy Your Own

1. Push this repository to GitHub
2. Connect your GitHub repo to Netlify
3. Point your domain to Netlify
4. Auto-deploy is enabled - any push to main branch will update the live site
